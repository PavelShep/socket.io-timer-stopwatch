Ноутбук и телефон должны быть подключены к одной сети wifi

Python 3.10.8

pip

pip install flask flask-socketio

python server.py

git bash

* Running on http://127.0.0.1:5000 - ноутбук (localhost:5000)
* Running on http://192.168.167.2:5000 - телефон, берётся ip wifi карты ноутбука с помощью ipconfig

Можно и на ноутбуке и на телефоне ввести 192.168.167.2:5000 и всё будет работать
